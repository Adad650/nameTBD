# Temple Escape

A 2D action-adventure game where you explore ancient temple rooms, solve puzzles, and fight enemies to escape. Built with Python and Pygame.

## Features

- Explore procedurally generated temple rooms
- Collect keys to unlock doors
- Fight enemies and avoid traps
- Smooth character animations
- Progressive difficulty as you advance through rooms
- Controller support

## Requirements

- Python 3.7+
- Pygame 2.0+

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Adad650/nameTBD
   cd temple-escape
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   
   Or install Pygame directly:
   ```bash
   pip install pygame
   ```

## How to Play

1. Run the game:
   ```zsh
   python3 main.py
   ```

2. Controls:
   - **WASD** or **Arrow Keys**: Move character
   - **E** or **Space**: Interact with objects (open chests, unlock doors)
   - **Left Mouse Button**: Attack
   - **ESC**: Pause/Quit game

## Gameplay

- Navigate through procedurally generated rooms
- Find and collect keys to unlock doors
- Defeat enemies to progress
- Avoid traps and hazards
- Each room gets progressively more challenging

## Project Structure

- `main.py` - Main game logic and entry point
- `assets/` - Game assets (sprites, images, sounds)
  - `death/` - Player death animation frames
  - `hurt/` - Player hurt animation frames
  - `idle/` - Player idle animation frames
  - `items/` - Game items (keys, chests)
  - `map/` - Background and tile images
  - `walking/` - Player walking animation frames
